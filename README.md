# Python购物商城系统 python235

本项目是一个基于Python语言的购物商城系统，包括源码和SQL文件，采用以下技术栈构建：

## 技术栈

- **后端：** Django框架
- **数据库：** MySQL
- **前端：** Vue.js

## 系统角色与功能

### 系统角色

- **用户：** 普通消费者角色，可以浏览商品、购买商品等。
- **管理员：** 系统管理角色，负责商品管理、订单管理以及用户管理等。

### 功能模块

- **商品浏览：** 展示商城中的商品列表。
- **商品详情：** 显示商品的详细信息。
- **个人信息管理：** 用户可以管理自己的个人资料。
- **购物车管理：** 添加、删除购物车中的商品。
- **订单管理：** 查看订单状态，进行订单支付等操作。
- **商品类型管理：** 管理商品分类。
- **订单支付：** 提供订单支付功能。

## 运行环境

确保以下环境配置以满足系统运行要求：

- Python版本：建议使用3.8以上版本
- Django版本：建议使用2.2或更高版本
- MySQL版本：建议使用5.7或更高版本
- Vue.js版本：建议使用2.x版本

## 部署步骤

1. 安装Python及Django环境。
2. 创建数据库，导入SQL文件以构建数据库结构。
3. 配置项目中的环境变量，如数据库连接等。
4. 运行Django开发服务器。
5. 构建Vue.js项目，并启动前端服务。

## 目录结构

```
 project/
 ├── backend/ # 后端Django项目目录
 ├── frontend/ # 前端Vue.js项目目录
 └── sql/ # 数据库SQL文件目录
```

## 核心亮点

- **稳定性：** 经过测试，系统运行稳定，无重大bug。
- **可扩展性：** 基于Django和Vue.js，易于维护和扩展功能。
